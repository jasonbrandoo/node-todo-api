# todos-api

My simple todo API using express and mongoose

### **How to use**

1.  npm install
1.  set up you mongo db in _app.js_
1.  npm start

## **API Endpoint**

### GET

- http://localhost:3000/todo

### POST

- http://localhost:3000/todo

### GET SINGLE ITEM

- http://localhost:3000/todo/todoId

### PATCH

- http://localhost:3000/todo/todoId

### DELETE

- http://localhost:3000/todo/todoId
